== Quickstart

. Install required collections:
+
[source,shell]
----
ansible-galaxy install -r requirements.yml
ansible-galaxy collection install -r requirements.yml
----

. Run the playbook:
+
[source,shell]
----
ansible-playbook -i hosts.yml flimsyCastle-setup.yml --ask-become-pass
----
